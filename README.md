## Getting Started
edit the env file
 
```env
NEXT_PUBLIC_DATABASE_URL="file:./path/to/sqlitedb"

NEXT_PUBLIC_GITHUB_ID="github-client-id"
NEXT_PUBLIC_GITHUB_SECRET="github-app-token"

NEXT_PUBLIC_GOOGLE_CLIENT_ID="google-client-id"
NEXT_PUBLIC_GOOGLE_CLIENT_SECRET="google-app-token"

NEXT_PUBLIC_SECRET="your-secret-key"
```

run the server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
